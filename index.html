Open Fabrication · MKRBOX

Modular Kinematic Replicator (MKRBOX)

A meter-cubed fabrication box designed to build almost anything it can reach — by coordinating one or more arms, modular tools, and a human operator who can supply parts on request.

Join the build Read the concept

1 m³ Workcell

A cubic meter envelope optimized for multi-axis reach, repeatability, and fast tool changes.

Tooling Library

Interchangeable end-effectors for assembly, fastening, inspection, wiring, and material handling.

Operator Requests

When a part is needed, the system asks the operator for delivery, keeping builds human-collaborative.

Open Source Stack

Hardware, software, and documentation in the open — iterate, fork, and contribute back.

  • Capabilities Multi-tool assembly, pick-and-place, routing, fastening, and inspection.
  • Control Layer Modular motion + task planning, with real-time operator prompts.
  • Build Philosophy Start simple, document everything, ship as a community of builders.
