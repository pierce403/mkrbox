Run locally

  • Clone the repo and open SIM.md for setup steps.
  • Expect an RTX-capable GPU and a setup flow that downloads NVIDIA sim dependencies.
  • You will accept NVIDIA terms during installation.
  • Start with the first level set in LEVELS.md.

Levels + repair fantasy

The game loop revolves around diagnosing failures, swapping modules, and refining the task plan. Each level unlocks new tools and introduces new constraints.

Browser preview vs full sim

The full sim runs on a GPU and streams to the browser. The website can host lightweight previews, but the real kinematic fidelity is server-side.
